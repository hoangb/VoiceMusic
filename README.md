Dependencies:
````
- Python 3.4.2 and up
- Flask
- spotipy
- SpeechRecognition
- pyttsx3
- pyaudio
- portaudio
````

Installation Commands:
````
- pip3 install Flask
- pip install spotipy
- pip install SpeechRecognition
- pip install pyttsx3
- pip install pyaudio
- pip install pyobjc
- brew install portaudio
````
